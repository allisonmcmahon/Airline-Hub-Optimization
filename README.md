# Airline-Hub-Optimization

## Problem
How can airlines choose hub airports to minimize total travel distance and improve network efficiency?

## Approach
- Used OpenFlights datasets (routes + airports)
- Modeled as a hub-and-spoke optimization problem
- Minimized weighted travel distance
- Selected optimal number of hubs using elbow method

## Key Results
- Optimal hub count ≈ 5
- Strategic hub selection significantly reduces network inefficiency
- Non-obvious hubs outperform intuitive choices
